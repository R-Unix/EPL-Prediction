Data Set

Dataset: The dataset contains the data of the game from the year 2000 to 2015. The dataset is in the csv format, ready to use within standard spreadsheet applications. The sheet contains data of different teams regarding home team shots, away team shots, home team fouls committed, away team fouls committed, home team offsides, away team offsides, number of yellow cards and red cards received and the full time result of home team and away team ie., wins, losses and draws.

Cleaning

The first step is to clean the dataset we will be using in the model. Data cleaning is important because it improves the data quality and increases the overall productivity. The main aim in this process is to identify and remove errors & duplicate data, in order to create a reliable dataset. By cleaning the data we make sure we use only those features which we need. The different points to consider while data cleaning are: ● Filling out missing values ● Removing rows with missing values ● Fixing error in the structure ● Reducing data for proper data handling

Visualization:

Data visualisation is an important aspect for understanding the data. It is the representation of data in the form of graphs or charts or different visual formats. It communicates the relation of different data in the form of visuals. With data visualisation different trends and patterns can be easily seen and understood which becomes difficult to understand through a spreadsheet. Data visualisation can be very helpful in determining correlations, examining networks, scheduling and analyzing value and risk

Training Data and Testing data

The most important thing to do before training and testing is the splitting of the data. The dataset can be splitted into two sets ie., train set and test set. The train set is used to train the model using any algorithm and the test set is used to evaluate the model. We need to remember one thing that we can not use the same data or the same set for training and testing. We compared various machine learning models’ accuracy for our dataset and used the one with highest accuracy for our project.

XgBoost

XGBoost is an algorithm that has recently been dominating applied machine learning and Kaggle competitions for structured or tabular data. XGBoost stands for eXtreme Gradient Boosting. XGBoost is a scalable, efficient, portable and improved version of a gradient boosting algorithm designed for computational speed and high performance. It is an open source library which can be downloaded and accessed from a variety of interfaces like: ● Command Line Interface (CLI). ● C++ (the language in which the library is written). ● Python interface as well as a model in scikit-learn. ● R interface as well as a model in the caret package. ● Julia.
